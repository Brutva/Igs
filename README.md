# Interactive Insurance Guide

Небольшое веб-приложение на **Next.js**, которое объясняет страхование простыми словами.  
Пользователь может пройти интерактивные сценарии, ответить на вопросы, получить результат и изучить теорию в понятном формате.

## Возможности

- главная страница с hero-блоком, прогрессом и карточками
- страница со сценариями
- отдельные страницы сценариев с пошаговыми вопросами
- страница с теорией
- квиз
- страница результата
- сохранение части прогресса в `localStorage`

## Стек

- Next.js
- React
- JavaScript
- Tailwind CSS
- Radix UI
- Lucide React
- Recharts

```bash
